How this tracker works
  • Local state. Progress is saved in your browser's localStorage. Different laptops see different state unless you sync.
  • Sync via git. Use Export to download tracker-state.json, commit it to the repo (e.g. under docs/), and teammates can Import it.
  • Blocked tasks subtract from overall progress and light the KPI card red. Unblock the task by clicking 🚫 again.
  • Notes per task are handy for capturing prompts and gotchas you'll paste into docs/learnings.md at the retro.
  • Print gives a clean snapshot for facilitator scoring against docs/rubric.md.
